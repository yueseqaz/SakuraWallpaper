# 🌸 SakuraWallpaper

A lightweight video & image wallpaper app for macOS.

[中文文档](README_CN.md)

## Features

- Set videos (MP4, MOV, GIF) or images (PNG, JPG, HEIC, WebP) as desktop wallpaper
- Multi-display support with different wallpapers per screen
- Recent wallpapers history
- Launch at login
- Pause/resume wallpaper
- Bilingual support (English / 中文)

## Installation

### Download

Download the latest `.dmg` from [Releases](../../releases) and drag SakuraWallpaper to Applications.

### Build from Source

```bash
git clone https://github.com/your-username/SakuraWallpaper.git
cd SakuraWallpaper
./build.sh
open build/SakuraWallpaper.app
```

## Usage

1. Click **Choose Wallpaper** to select a video or image file
2. Use the screen dropdown to switch between displays
3. Click **Apply to All** to set the same wallpaper on all screens
4. Right-click the 🌸 icon in the status bar for more options

## Supported Formats

| Video | Image |
|-------|-------|
| MP4 | PNG |
| MOV | JPG / JPEG |
| GIF | HEIC |
| M4V | WebP |
| | BMP / TIFF |
