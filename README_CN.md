# 🌸 樱花壁纸 (SakuraWallpaper)

macOS 轻量级视频和图片壁纸应用。

[English](README.md)

## 功能特性

- 将视频 (MP4, MOV, GIF) 或图片 (PNG, JPG, HEIC, WebP) 设置为桌面壁纸
- 多显示器支持，可为每个屏幕设置不同壁纸
- 最近使用壁纸历史
- 开机自启动
- 暂停/恢复壁纸
- 中英文双语支持

## 安装

### 下载

从 [Releases](../../releases) 下载最新的 `.dmg` 文件，将 SakuraWallpaper 拖入应用程序文件夹。

### 从源码构建

```bash
git clone https://github.com/your-username/SakuraWallpaper.git
cd SakuraWallpaper
./build.sh
open build/SakuraWallpaper.app
```

## 使用方法

1. 点击 **选择壁纸** 选择视频或图片文件
2. 使用屏幕下拉框切换不同显示器
3. 点击 **应用到全部** 将同一壁纸设置到所有屏幕
4. 右键点击状态栏的 🌸 图标查看更多选项

## 支持格式

| 视频 | 图片 |
|------|------|
| MP4 | PNG |
| MOV | JPG / JPEG |
| GIF | HEIC |
| M4V | WebP |
| | BMP / TIFF |

## 系统要求

- macOS 12.0 或更高版本
